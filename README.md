# Task-02 Markov Chains

This project generates text using a simple Markov Chain model.

## Files
- markov_chain.ipynb → main code
- data.txt → training data
- generated_output.txt → output text
- requirements.txt → dependencies

## How it works
It predicts next word based on previous word probability.
